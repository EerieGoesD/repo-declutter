Finds API keys, tokens, and key files committed to git, which means they reach GitHub once pushed. This scan is read only. It changes nothing and never shows the secret itself.
How to read the results. Critical and High match distinctive key formats, so they are rarely wrong. Review casts a wide net on purpose, so give those a look yourself. Obvious fakes and test lines are skipped. The scan only knows the formats it looks for, so an unusual key can slip through, and it cannot tell whether a key is meant to be public. A clean result is a good sign, not proof.
